# 体育管理系统

体育管理系统

#### 介绍

技术栈:
基于vue+mysql+Spring+mybatis+tomcat的管理系统

#### 功能:

用户端和管理端:  

体育馆管理系统 赛事列表管理 器材租借管理 管理 系统管理(用户、角色、菜单、公告) 器材列表 场地管理等功能
